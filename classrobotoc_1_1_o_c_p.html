Construct the optiaml control problem. The switching time optimization (STO) algorithm can be enabled with this constructor. Moreover, the discretization method of the optimal control problem is fixed to DiscretizationMethod::PhaseBased.

Parameters
[in]robotRobot model.
[in]costShared ptr to the cost function.
[in]constraintsShared ptr to the constraints.
[in]sto_costShared ptr to the STO cost function.
[in]sto_constraintsShared ptr to the STO constraints.
[in]TLength of the horzion. Must be positive.
[in]NNumber of the discretization grids of the horizon except for the discrete events. Must be positive.
[in]max_num_each_discrete_eventsMaximum possible number of the each discrete events on the horizon. Must be non-negative. Default is 0.
